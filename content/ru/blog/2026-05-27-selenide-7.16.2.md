### Параметры для Selenide MCP server {#selenide-mcp-parameters}

Теперь нашему MCP серверу можно задавать разные там всякие параметры. Например, URL грида или размер окна браузера.

Параметры задаются в том же самом файлике `.mcp.json`, в котором вы обычно прописываете MCP:

```json
{
  "mcpServers": {
    "selenide-mcp": {
      "command": "npx",
      "args": [
        "selenide-mcp",
        "--browser=chrome",
        "--remote=http://selenium-hub:4444/wd/hub"
      ]
    }
  }
}
```

Полный список параметров можно найти в [README](https://github.com/selenide/selenide/blob/main/modules/mcp/README.md).

### Мелочи при скачивании файлов {#file-download-improvements}

Кое-что по мелочам подправили:

* Если скачалось несколько подходящих файлов, берём последний (до этого мы брали самый старый файл - это вышло случайно)
* При скачивании методом CDP: Пофиксили случай, когда событие "downloadWillBegin" случается ПОСЛЕ "downloadProgress" (это нелогично, но иногда случается)
* При скачивании методом FOLDER: Добавили инфу в логи при скачивании файлов

См. [PR 3333](https://github.com/selenide/selenide/pull/3333) и [PR 3332](https://github.com/selenide/selenide/pull/3332).
