### Parameters for Selenide MCP server {#selenide-mcp-parameters}

You can now pass all sorts of parameters to our MCP server. For example, the grid URL or the browser window size.

The parameters go in the same `.mcp.json` file where you usually configure MCP:

```json
{
  "mcpServers": {
    "selenide-mcp": {
      "command": "npx",
      "args": [
        "selenide-mcp",
        "--browser=chrome",
        "--remote=http://selenium-hub:4444/wd/hub"
      ]
    }
  }
}
```

The full list of parameters is in the [README](https://github.com/selenide/selenide/blob/main/modules/mcp/README.md).

Thanks to [Mikhail Sidelnikov](https://github.com/sidelnikovmike) for [PR 3323](https://github.com/selenide/selenide/pull/3323).

### Small things for file downloads {#file-download-improvements}

A few minor fixes:

* If several matching files were downloaded, we now pick the latest one (until now we were picking the oldest file — that happened by accident)
* For the CDP download method: fixed the case when the "downloadWillBegin" event arrives AFTER "downloadProgress" (it's illogical, but it does happen sometimes)
* For the FOLDER download method: added more info to the logs when downloading files

See [PR 3333](https://github.com/selenide/selenide/pull/3333) and [PR 3332](https://github.com/selenide/selenide/pull/3332).
